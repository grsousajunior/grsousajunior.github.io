Modelo 3D de uma geometria molecular trigonal bipiramidal a partir de um arranjo tetraédrico. O modelo foi preparado usando o Avogadro 1.2.0 (versão modificada pela equipe do pacote computacional ORCA e compatível com o ORCA 4.2) a partir do metano, com a minimização da energia feita usando o campo de força UFF (em inglês). Os orbitais foram obtidos a partir de cálculo single point usando o método composto B97-3c com localização de orbitais (em inglês) usando o método Foster-Boys (em inglês), disponíveis no pacote computacional ORCA 4.2.1. O modelo 3D foi construído usando o ChimeraX 1.2, modificando/ocultando alguns átomos para coincidir com a geometria molecular desejada, além das superfícies, geradas a partir dos orbitais moleculares localizados, com isovalor de 0,1.
